Pindah Perangkat
Langkah - Langkah Pindah Perangkat:

1. Klik tombol UPLOAD di halaman ini

2. Tunggu proses upload data ke server

3. Logout akun Barangku di perangkat ini

4. Login akun Barangku di perangkat baru

5. Masuk ke halaman Pindah Perangkat

6. Klik tombol DOWNLOAD di halaman ini

7. Tunggu proses download data dari server

8. Data di perangkat lama berhasil dipindahkan

*data yang telah diupload ke server hanya bertahan selama 2 x 24 jam. Pastikan anda download data tersebut di perangkat baru sebelum terhapus secara otomatis.
